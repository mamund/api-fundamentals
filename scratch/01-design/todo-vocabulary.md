# TO-DO Vocabulary

## Propertes
- ID [R] _string_
- title [R] _string_
- status [R] see enumerations, defaults to `pending`

## Actions
- list
- filter (title[partial], status[enum])
- add (title, status)
- update (id, title, status)
- markCompleted (id)

## Enumerations
- status: [pending, working, completed]
