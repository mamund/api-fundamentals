TO-DO List API

A simple command-line app to help people keep track of things to do.

  • Semantic Descriptors
    • collection (semantic), List of TODO records
    • dateDue (semantic), Due date for an existing TODO record (simple: YYYY-MM-DD)
    • doCreate (unsafe), Create a new TODO record
    • doDue (idempotent), Update the due date value of an existing TODO record.
    • doStatus (idempotent), Update the status value of an existing TODO record.
    • doUpdate (idempotent), Update an existing TODO record
    • goFilter (safe), Filter the list of person records
    • goHome (safe), Go to the Home resource
    • goItem (safe), Go to a single TODO record
    • goList (safe), Go to the list of TODO records
    • home (semantic), Home (starting point) of the TODO service
    • id (semantic), Id of the record.
    • item (semantic), Single TODO record
    • status (semantic), Status of the person record (pending, completed)
    • title (semantic), The title of the job.
    • todo (semantic), The properties of a TODO record
